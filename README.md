# Bauman Doomlike

Учебный проект по дисциплине "программирование классов и шаблонов": упрощённый аналог Doom на C++ с псевдо-3D сценой через raycasting.

## Стек

- `C++20`
- `CMake`
- `SDL3`
- `ENet`

## Быстрый старт

```bash
cmake -S . -B build
cmake --build build
./build/bauman_doomlike
```

По умолчанию зависимости подтягиваются через `FetchContent`. Это упрощает запуск на macOS, Windows и Linux без отдельной ручной установки SDL3 и ENet.

При запуске откроется стартовое меню:

- `1` - тестовый режим, как раньше;
- `2` - создать мультиплеерную игру в локальной сети;
- `3` - подключиться к хосту по IP.

Для LAN-мультиплеера один игрок выбирает режим хоста, второй запускает игру на другом компьютере в той же сети, выбирает подключение и вводит IP хоста. Используемый порт: `27015`.

## Управление в текущем каркасе

- `W/S` - движение вперёд/назад
- `A/D` - стрейф влево/вправо
- `←/→` - поворот
- `Space` - выстрел
- `R` - рестарт после окончания матча
- `Esc` - выход
