# Bauman Doomlike

Учебный проект по дисциплине "программирование классов и шаблонов": упрощённый аналог Doom на C++ с псевдо-3D сценой через raycasting.

## Стек

- `C++20`
- `CMake`
- `SDL3`
- `ENet`

## Быстрый старт

```bash
cmake -S . -B build
cmake --build build
./build/bauman_doomlike
```

По умолчанию зависимости подтягиваются через `FetchContent`. Это упрощает запуск на macOS, Windows и Linux без отдельной ручной установки SDL3 и ENet.

## Управление в текущем каркасе

- `W/S` - движение вперёд/назад
- `A/D` - стрейф влево/вправо
- `←/→` - поворот
- `Space` - выстрел
- `R` - рестарт после окончания матча
- `Esc` - выход
